# Website – OPOR (One Place One Result)

Dieser Ordner enthält alle Dateien, die für den Aufbau, das Design und die Funktionen der OPOR-Webseite notwendig sind.  
Die Webseite dient als Benutzeroberfläche für den Abiturnotenrechner sowie für zusätzliche Funktionen wie Login, Profil und weitere Seiten.
